# OAI List
This repo serves as a problem archive for regional OAI and IOAI. Problems are all organized in folders which you can quickly download and use.

Small dataset is pre-downloaded into the folder, large dataset like images still require downloading manually when running the notebook.

[Website for easier browsing & downloading](https://yunnokon.github.io/oai-list/)
